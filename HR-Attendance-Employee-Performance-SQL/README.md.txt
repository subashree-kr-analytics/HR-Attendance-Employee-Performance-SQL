# 👨‍💼 HR Attendance & Employee Performance Analytics (SQL)

## 📌 Project Description
This project is a SQL-based HR Analytics system that helps track employee attendance and analyze performance.  
It provides insights using SQL queries to evaluate attendance patterns and employee performance.

---

## ⚙️ Features
- 📅 Employee attendance tracking
- 📊 Performance analysis using SQL queries
- 📈 Department-wise reports
- 👨‍💼 Employee record management
- 📉 Attendance summary reports

---

## 🧑‍💻 Technologies Used
- SQL (MySQL / PostgreSQL / SQLite)

---

## 🚀 How to Run
1. Open any SQL database tool (MySQL / SQLite / PostgreSQL)
2. Import or open `queries.sql`
3. Run queries step by step
